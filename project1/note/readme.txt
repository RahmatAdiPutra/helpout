Yang harus dilakukan setelah download project ini :

1. buat database baru dengan nama project1
2. import file database project1.sql yang ada didalam folder note ke database yang baru dibuat pada point 1
3. buka cmd/commandline/console
4. masuk kedalam project1 melalui cmd/commandline/console
5. setelah itu ketikan perintah "composer install" enter
6. tunggu sampai selesai
7. setelah selesai, cek file '.env' ada atau tidak didalam folder project1
8. jika tidak ada, buat file baru '.env'
9. copy isi file 'env.txt' yang ada di folder note ke dalam file '.env'
